How to Play

This is a simple game called asteroids only available on PC!

Press enter to start the game.Move the blue triangle using up/right/left,and fire a bullet using space.
The score shows at the right-up corner of the screen.The yellow ones are coins,eat them to get more scores.

Thanks to Doug McInnes for creating this awesome game!
